Both the Python and Node servers are hosted on Heroku, and the backend code is available at https://github.com/Goobber33/Chatbot-Back-End. The application uses OpenAI's powerful language models to provide a conversational interface that can answer a wide range of user queries, and also incorporates weather data from the OpenWeather API and stock market data from the Alpha Advantage API. â˜ï¸ğŸ“ˆ
